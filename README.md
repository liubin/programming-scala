# Scala 编程

## 为什么编写这本书

- 市面上没有太多的中文书，聊聊基本还都是翻译的，质量一般；入门的话会经常看到一些让人莫名其妙的翻译而且没有给定英语原文。
- 顺便加深学习。
- Scala本身挺有意思的。
